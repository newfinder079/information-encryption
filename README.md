# 魑魅魍魉加密

## 项目简介
魑魅魍魉加密是一个简单的加密和解密工具，允许用户输入明文并使用口令进行加密，或将密文解密回明文。该应用不需要联网即可工作，确保用户的隐私和数据安全。

## 文件结构
```
chimeiwangliang
├── index.html       # 应用的主HTML文件
├── style.css        # 应用的样式定义
├── app.js           # 应用的JavaScript逻辑
└── README.md        # 项目的文档
```

## 功能
- **加密**：用户可以输入明文和口令，生成密文。
- **解密**：用户可以输入密文和口令，恢复明文。
- **自检**：运行自检功能，确保加密和解密过程正常。

## 使用方法
1. 打开 `index.html` 文件。
2. 在“加密”部分输入要加密的文本和口令，点击“加密”按钮。
3. 在“解密”部分输入密文和口令，点击“解密”按钮。
4. 使用“复制密文”按钮将生成的密文复制到剪贴板。
5. 点击“运行自检”按钮进行自检。

## 注意事项
- 请妥善保管您的口令，避免泄露。
- 本应用不存储用户输入的数据，所有操作均在本地进行。
